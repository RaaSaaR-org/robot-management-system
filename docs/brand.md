# üé® **2. Logo System**

## **Logo Concept**

The RoboMindOS logo combines:

- **A neural-inspired symbol** (representing intelligence)
- **A robot silhouette or circuit-like glyph** (representing robotics)
- **OS-style geometric precision** (representing the software platform)

## **Logo Variations**

- **Primary Logo:** Icon + Wordmark ("RoboMindOS")
- **Secondary Logo:** Icon only
- **Monochrome Version:** White or black for dark/light mode
- **App Icon:** Square format, simplified symbol

## **Logo Usage Rules**

- Minimum clear space: height of the letter "R"
- Avoid distortion, rotations, filters, and busy backgrounds
- Use brand colors only
